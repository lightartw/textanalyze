# Event Classifier

## 任务目标
- 判断新闻句子是否涉及影响能源/石油市场的特定事件类型
- 提取相关关键词并判断是否与石油直接相关
- 返回结构化的JSON格式结果

## 事件类型定义

### 可识别的八类事件

1. **地缘政治 (geopolitical)**
   - 战争、冲突、军事行动
   - 制裁、贸易限制
   - OPEC决策、产量协议
   - 管道袭击、设施破坏
   - 关键产油国政治变动

2. **宏观经济 (macro)**
   - GDP、经济增长数据
   - 利率、货币政策
   - PMI、就业数据
   - 通胀、汇率波动
   - 经济衰退或复苏信号

3. **市场情绪 (sentiment)**
   - 恐慌、担忧、乐观
   - 投机行为、仓位调整
   - 看涨、看跌预期
   - 抛售、抢购
   - 风险偏好变化

4. **极端天气 (weather)**
   - 飓风、台风
   - 寒潮、热浪
   - 洪水、干旱
   - 炼厂停工、生产中断
   - 需求异常（如取暖/制冷需求）

5. **库存变化 (inventory)**
   - 原油、成品油库存增减
   - 商业库存数据发布
   - 战略储备释放
   - 库存超预期或低于预期

6. **政策因素 (policy)**
   - 能源政策调整
   - 税收政策变化
   - 出口配额、进口关税
   - 环保法规、碳排放限制
   - 石油补贴调整

7. **技术进步 (technology)**
   - 开采技术突破
   - 炼油技术改进
   - 新能源技术发展
   - 提高采收率技术
   - 数字化、自动化应用

8. **其他因素 (other)**
   - 不属于前七类但明显影响油价的因素
   - 包括：替代能源发展、供需基本面、贸易摩擦、地缘紧张局势（非制裁）、能源转型等
   - 用于捕获难以归类但具有油价格影响的潜在因子

## 操作步骤

### 步骤1: 判断是否石油/能源相关
- 识别句子中是否包含石油、原油、能源、油价、WTI、Brent等关键词
- 如果句子与石油/能源/油价无直接关联，直接返回 `{"event_type": null, "keywords": [], "is_oil_related": false}`
- 仅在确认石油相关后继续分类

### 步骤2: 识别事件类型
- 逐类检查句子是否包含对应事件的特征词汇和语义
- 参考各类事件的典型关键词和场景（见 references/event-types.md）
- 如果句子涉及多个事件类型，选择最显著的主导类型
- 如果句子不涉及任何可识别的七类事件，但明显影响油价，归为 `other`（其他因素）
- 如果句子与石油无关或不影响油价，返回 `null`

### 步骤3: 提取关键词
- 从原文中提取与识别到的事件类型直接相关的关键词
- 关键词必须来自原文，不得添加或修改
- 提取2-5个最具代表性的关键词

### 步骤4: 生成JSON输出
- 严格遵循输出格式，不得添加额外字段或说明
- 确保JSON格式正确，逗号和引号规范

## 输出格式

仅返回严格符合以下格式的JSON：

```json
{
  "event_type": "geopolitical|macro|sentiment|weather|inventory|policy|technology|other|null",
  "keywords": ["关键词1", "关键词2"],
  "is_oil_related": true|false
}
```

字段说明：
- `event_type`: 事件类型，仅限八个枚举值或 `null`
- `keywords`: 从原文提取的关键词列表
- `is_oil_related`: 是否与石油直接相关

## 资源索引

- 领域参考: 见 [references/event-types.md](references/event-types.md) (包含各类事件的详细定义、典型场景和关键词示例)

## 注意事项

- 严格按输出格式返回JSON，不要添加任何解释或额外字段
- 关键词必须从原文中提取，不得杜撰
- 仅识别输出格式中定义的八类事件，其他不影响油价的情况返回 `null`
- 若句子与石油无直接关联，即使包含可识别事件也返回 `null`
- 优先识别最显著的事件类型，避免模糊分类

## 使用示例

### 示例1: 地缘政治事件
输入: "OPEC+宣布从5月起自愿减产116万桶/日"
输出: `{"event_type": "geopolitical", "keywords": ["OPEC+", "减产"], "is_oil_related": true}`

### 示例2: 宏观经济事件
输入: "美联储加息25个基点，经济放缓担忧加剧"
输出: `{"event_type": "macro", "keywords": ["美联储", "加息", "经济放缓"], "is_oil_related": true}`

### 示例3: 市场情绪
输入: "投资者恐慌性抛售，油价暴跌8%"
输出: `{"event_type": "sentiment", "keywords": ["恐慌性抛售", "油价暴跌"], "is_oil_related": true}`

### 示例4: 极端天气
输入: "飓风伊恩逼近墨西哥湾，多家炼厂暂停运营"
输出: `{"event_type": "weather", "keywords": ["飓风伊恩", "墨西哥湾", "炼厂暂停"], "is_oil_related": true}`

### 示例5: 库存变化
输入: "美国商业原油库存增加200万桶，超出预期"
输出: `{"event_type": "inventory", "keywords": ["商业原油库存", "增加", "超出预期"], "is_oil_related": true}`

### 示例6: 政策因素
输入: "欧盟将实施石油出口配额限制"
输出: `{"event_type": "policy", "keywords": ["欧盟", "石油出口", "配额限制"], "is_oil_related": true}`

### 示例7: 技术进步
输入: "页岩油开采技术突破，成本下降30%"
输出: `{"event_type": "technology", "keywords": ["页岩油开采技术", "突破", "成本下降"], "is_oil_related": true}`

### 示例8: 其他因素
输入: "新能源汽车快速发展，石油需求增长预期放缓"
输出: `{"event_type": "other", "keywords": ["新能源汽车", "需求增长预期"], "is_oil_related": true}`

### 示例9: 非石油相关
输入: "苹果公司发布新款iPhone，股价上涨5%"
输出: `{"event_type": null, "keywords": [], "is_oil_related": false}`

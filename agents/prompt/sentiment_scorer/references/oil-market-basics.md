## 概览
本参考文档提供石油市场的基础概念和机制，用于理解事件对油价的潜在影响路径。国际油价主要由WTI（西德克萨斯中质原油）和Brent（布伦特原油）两大基准油价主导。

## 供需机制

### 供给端因素
- **OPEC+产油国**：沙特、俄罗斯等主导国，通过产量配额调控供给
- **非OPEC供给**：美国页岩油、加拿大油砂、巴西深海油田等
- **供应中断**：战争、制裁、自然灾害、设施事故等导致供给减少
- **产能调整**：主动减产/增产、新油田投产、老旧油田衰竭

### 需求端因素
- **经济周期**：经济增长带动能源需求，衰退导致需求萎缩
- **季节性变化**：夏季出行高峰、冬季取暖需求
- **替代能源**：新能源发展对石油需求的长期替代
- **库存水平**：商业库存、战略储备的释放/补库

## 主要参与者

### 供给侧
- **OPEC**：石油输出国组织（13个成员国）
- **OPEC+**：OPEC及其盟友（以俄罗斯为代表）
- **美国页岩油商**：高弹性、成本敏感的供给主体
- **国家石油公司**：沙特阿美、中石油、委内瑞拉PDVSA等

### 需求侧
- **进口国**：中国、印度、欧盟、日本等主要消费国
- **炼厂**：将原油加工成成品油的中游环节
- **终端用户**：交通运输、工业、化工、发电等行业

### 影响者
- **IEA**：国际能源署，协调成员国战略储备
- **EIA**：美国能源信息署，发布权威市场数据
- **投机资金**：对冲基金、CTA等短期价格波动参与者

## 市场指标

### 供给指标
- OPEC+产量配额与实际产量
- 美国原油产量（EIA周报）
- 产量意外中断报告
- 钻机数量（Baker Hughes数据）

### 需求指标
- 全球原油消费增速（IEA/EIA月报）
- 主要经济体PMI数据
- 成品油需求指标（如汽油库存消费比）
- 航运、公路运输活跃度

### 库存指标
- OECD商业库存
- 美国EIA/API周度库存
- 战略储备释放/补库动向
- 库存季节性调整后的超预期/低于预期

### 宏观指标
- 美元指数（美元计价资产）
- 全球经济增长预期（IMF/世界银行）
- 地缘政治风险指数
- VIX恐慌指数

## 价格影响因素

### 短期因素（天~周级）
- 库存周度数据波动
- 地缘突发事件
- 极端天气影响
- 投机情绪变化
- 美元短期波动

### 中期因素（月~季度级）
- OPEC+产量调整
- 美国页岩油产能变化
- 季节性需求变化
- 主要经济体经济数据
- 制裁政策变化

### 长期因素（年~十年级）
- 新能源替代趋势
- 全球能源转型政策
- 石油勘探开发投资
- 需求峰值的预期

## 事件影响传导路径

```
事件发生 → 识别供给侧/需求侧影响 → 评估规模和持续时间 → 对比历史事件 → 量化情绪极性 → 量化冲击强度
```

### 典型传导示例
- **地缘冲突**：中断供给 → 紧张局势升级 → 供给减少预期 → 利多情绪 → 高强度冲击
- **经济衰退**：需求预期下降 → 消费缩减 → 库存积累 → 利空情绪 → 中高强度冲击
- **OPEC减产**：主动缩减供给 → 市场平衡收紧 → 供给曲线左移 → 利多情绪 → 中度冲击

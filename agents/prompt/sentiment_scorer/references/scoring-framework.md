## 概览
本参考文档提供情绪极性和冲击强度的系统化评分框架，包含详细的决策流程和典型场景处理指南。

## 评分维度定义

### 情绪极性(Sentiment)
**定义**：事件对油价的利多或利空方向
**范围**：-1.0（极度利空）→ +1.0（极度利好）
**核心原则**：
- 供给减少 / 需求增加 → 正极性
- 供给增加 / 需求下降 → 负极性
- 情绪极性是方向性判断，不涉及影响幅度

### 冲击强度(Intensity)
**定义**：事件对油价的冲击程度和持续时间
**范围**：0.0（无影响）→ 1.0（重大冲击）
**核心原则**：
- 评估维度：影响规模、持续时间、市场关注度
- 历史对比：参考历史事件的冲击强度
- 冲击强度是幅度判断，与情绪方向独立

### 置信度(Confidence)
**定义**：评分结果的可信度和可靠性
**范围**：0.0（极低可信）→ 1.0（极高可信）
**核心原则**：
- 多维度综合评估：信息完整性、历史匹配度、多维度一致性等
- 透明性要求：必须提供各维度评分和加权逻辑
- 动态调整：发现异常或不一致时动态降低置信度

## 情绪极性评分流程

### Step 1: 识别事件类型
```
事件分类树：
├── 供给端
│   ├── 主动减产/停产 → 利多(+)
│   ├── 被动中断（战争/制裁/灾害） → 利多(+)
│   ├── 主动增产/复产 → 利空(-)
│   └── 新产能投产 → 利空(-)
├── 需求端
│   ├── 经济增长 → 利多(+)
│   ├── 经济衰退 → 利空(-)
│   ├── 季节性需求旺季 → 利多(+)
│   └── 季节性需求淡季 → 利空(-)
├── 地缘政治
│   ├── 威胁供给中断 → 利多(+)
│   ├── 缓解供给担忧 → 利空(-)
│   └── 不确定性上升（双向） → 视具体情况
├── 宏观经济
│   ├── 美元走强 → 利空(-)
│   ├── 美元走弱 → 利多(+)
│   ├── 通胀上升 → 利多(+)
│   └── 通胀下降 → 利空(-)
└── 其他
    ├── 库存下降 → 利多(+)
    └── 库存上升 → 利空(-)
```

### Step 2: 评估极性幅度
**0.1~0.3（轻微）**：
- 常规库存数据小幅超预期
- 钻机数量小幅波动
- 单日天气影响
- 技术性交易因素

**0.4~0.6（中度）**：
- OPEC常规产量调整
- EIA/API库存大幅超预期
- 主要经济体PMI数据显著变化
- 美元指数大幅波动

**0.7~0.9（显著）**：
- OPEC+重大减产/增产决定
- 主要产油国制裁/解除制裁
- 重大地缘政治事件（威胁供给）
- 全球经济衰退/繁荣预警

**1.0（极端）**：
- 全面禁运/断供
- 大规模战争爆发
- 全球性危机（金融危机、疫情）
- OPEC完全解散或全面限产

### Step 3: 复合事件处理
当事件同时包含利多和利空因素时：
1. 分别评估各因素的极性和权重
2. 计算加权平均极性
3. 考虑主导因素方向

示例："OPEC减产50万桶/日，但美国经济数据恶化"
- OPEC减产：+0.5（权重40%）
- 美国经济：-0.4（权重60%）
- 综合极性：0.5×0.4 + (-0.4)×0.6 = -0.04 → 修正为 -0.1（小幅利空）

## 冲击强度评分流程

### Step 1: 评估影响规模
**全球级影响（占比≥5%）**：
- 影响全球300万桶/日以上供给
- 影响全球需求增速超过1%
- 基准强度：0.8+

**区域级影响（占比1-5%）**：
- 影响区域100-300万桶/日供给
- 影响主要经济体需求
- 基准强度：0.5-0.8

**局部级影响（占比<1%）**：
- 影响单国或单区域供给
- 影响局部需求
- 基准强度：0.2-0.5

**边际级影响（占比<0.1%）**：
- 影响短期供需平衡
- 无实质性规模影响
- 基准强度：0.0-0.2

### Step 2: 评估持续时间
**长期影响（6个月以上）**：
- 结构性变化（如产能关停、长期制裁）
- 基准强度调整：+0.1

**中期影响（1-6个月）**：
- 周期性变化（如季节性需求、季度调整）
- 基准强度调整：基准值

**短期影响（1个月以内）**：
- 临时性冲击（如短期停产、突发事件）
- 基准强度调整：-0.1

**超短期影响（1周以内）**：
- 日内波动、短暂扰动
- 基准强度调整：-0.2

### Step 3: 评估市场关注度
**主导性因素**：
- 市场完全聚焦该事件
- 所有分析报告都在讨论
- 基准强度调整：+0.1

**主要因素**：
- 市场广泛关注
- 多数分析师提及
- 基准强度调整：基准值

**次要因素**：
- 市场有一定关注
- 部分分析师提及
- 基准强度调整：-0.05

**边际因素**：
- 市场关注度低
- 仅有少数交易者关注
- 基准强度调整：-0.1

### Step 4: 综合评分计算
```
冲击强度 = 基准强度 + 时间调整 + 关注度调整
```

约束条件：
- 最终强度范围：0.0 ≤ intensity ≤ 1.0
- 调整后若超出范围，边界取整（如计算1.2 → 1.0）

## 典型场景决策树

### 场景1：供给中断
```
供给中断规模？
├── >300万桶/日（全球级）
│   ├── 持续时间 >6个月 → intensity = 0.9+
│   ├── 持续时间 1-6个月 → intensity = 0.8-0.9
│   └── 持续时间 <1个月 → intensity = 0.7-0.8
├── 100-300万桶/日（区域级）
│   ├── 持续时间 >6个月 → intensity = 0.6-0.7
│   ├── 持续时间 1-6个月 → intensity = 0.5-0.6
│   └── 持续时间 <1个月 → intensity = 0.4-0.5
└── <100万桶/日（局部级）
    ├── 持续时间 >6个月 → intensity = 0.3-0.4
    ├── 持续时间 1-6个月 → intensity = 0.2-0.3
    └── 持续时间 <1个月 → intensity = 0.1-0.2
```

### 场景2：需求变化
```
需求变化幅度？
├── 需求萎缩 >5%（全球衰退）
│   ├── 全球性危机 → sentiment = -0.9, intensity = 0.85
│   └── 区域性衰退 → sentiment = -0.7, intensity = 0.6
├── 需求萎缩 1-5%（经济放缓）
│   ├── 主要经济体 → sentiment = -0.5, intensity = 0.4
│   └── 区域经济体 → sentiment = -0.3, intensity = 0.25
├── 需求增长 1-5%（经济复苏）
│   ├── 主要经济体 → sentiment = 0.4, intensity = 0.35
│   └── 区域经济体 → sentiment = 0.25, intensity = 0.2
└── 需求增长 >5%（繁荣期）
    └── 全球性繁荣 → sentiment = 0.7, intensity = 0.55
```

### 场景3：OPEC+产量调整
```
调整规模？
├── >200万桶/日（重大调整）
│   ├── 减产 → sentiment = 0.7, intensity = 0.7
│   └── 增产 → sentiment = -0.7, intensity = 0.7
├── 100-200万桶/日（较大调整）
│   ├── 减产 → sentiment = 0.6, intensity = 0.6
│   └── 增产 → sentiment = -0.6, intensity = 0.6
├── 50-100万桶/日（常规调整）
│   ├── 减产 → sentiment = 0.4, intensity = 0.4
│   └── 增产 → sentiment = -0.4, intensity = 0.4
└── <50万桶/日（小幅调整）
    ├── 减产 → sentiment = 0.25, intensity = 0.25
    └── 增产 → sentiment = -0.25, intensity = 0.25
```

### 场景4：地缘政治事件
```
威胁等级？
├── 全面战争/全面禁运（极度威胁）
│   ├── 已发生 → sentiment = 0.95, intensity = 0.95
│   └── 高度预期 → sentiment = 0.8, intensity = 0.7
├── 区域冲突/制裁（中度威胁）
│   ├── 已发生 → sentiment = 0.7, intensity = 0.65
│   └── 预期上升 → sentiment = 0.5, intensity = 0.4
└── 局部冲突/有限制裁（低度威胁）
    ├── 已发生 → sentiment = 0.4, intensity = 0.35
    └── 预期上升 → sentiment = 0.25, intensity = 0.2
```

## 置信度评分框架

### 置信度评估维度

置信度由以下5个维度综合评估，每个维度得分范围0.0-1.0：

#### 1. 信息完整性 (Information Completeness)
**定义**：事件描述的详细程度和关键信息齐全度

**评分标准**：
- 1.0：信息极其完整，包含所有关键要素（涉及主体、影响规模、时间维度、具体数据）
- 0.8：信息较完整，包含大部分关键要素
- 0.6：信息一般，部分关键要素缺失
- 0.4：信息有限，关键要素不足
- 0.2：信息模糊，仅有事件概述
- 0.0：信息严重缺失

**示例**：
- "沙特宣布从下月起自愿减产100万桶/日，持续6个月" → 0.9
- "OPEC可能减产" → 0.3

#### 2. 历史事件匹配度 (Historical Event Match)
**定义**：当前事件与历史相似事件的匹配程度

**评分标准**：
- 1.0：高度相似，有完全可比的历史事件（类型、规模、时间维度相近）
- 0.8：相似度高，有可参照的历史事件（主要特征相似）
- 0.6：中度相似，有部分参照的历史事件
- 0.4：相似度低，缺乏直接参照的历史事件
- 0.2：几乎无参照，历史案例不足
- 0.0：无参照

**示例**：
- "沙特减产100万桶/日"（参考2020年OPEC减产）→ 0.9
- "新兴地缘冲突"（历史无直接参照）→ 0.4

#### 3. 多维度一致性 (Multi-dimensional Consistency)
**定义**：供给/需求/地缘/宏观多维度评估的一致性

**评分标准**：
- 1.0：所有维度评估完全一致
- 0.8：大部分维度评估一致（≥75%）
- 0.6：部分维度评估一致（≥50%）
- 0.4：少数维度评估一致（≥25%）
- 0.2：维度评估存在矛盾
- 0.0：维度评估严重不一致

**示例**：
- "OPEC减产100万桶/日"（供给端明确，需求/地缘/宏观无冲突）→ 0.9
- "地缘冲突但需求预期增长"（供给/需求方向矛盾）→ 0.5

#### 4. 市场预期偏离度 (Market Expectation Deviation)
**定义**：事件与市场预期的一致程度（一致则置信度高）

**评分标准**：
- 1.0：事件与市场预期高度一致，已被充分预期
- 0.8：事件与市场预期基本一致，轻微偏离
- 0.6：事件与市场预期存在一定偏离
- 0.4：事件与市场预期偏离较大
- 0.2：事件与市场预期严重背离（超预期）
- 0.0：完全无法判断

**注意**：超预期事件可能产生更大的市场影响，但评估不确定性增加，导致置信度降低。

**示例**：
- "市场已充分预期OPEC减产100万桶/日，会议正式公布" → 0.9
- "市场预期维持产量，OPEC意外宣布减产100万桶/日" → 0.5

#### 5. 评估者经验权重 (Evaluator Experience Weight)
**定义**：评估者对该类型事件的经验程度和知识掌握度

**评分标准**：
- 1.0：经验丰富，对该类型事件有深入理解
- 0.8：经验较丰富，对该类型事件有较好理解
- 0.6：经验一般，对该类型事件有基本理解
- 0.4：经验有限，对该类型事件理解不足
- 0.2：经验极少，对该类型事件了解有限
- 0.0：无经验

**注意**：此维度反映评估者对特定类型事件的熟悉程度，并非绝对能力评估。

### 置信度加权计算

#### 标准权重配置
```
confidence = (信息完整性 × 0.25) + (历史事件匹配度 × 0.25) + (多维度一致性 × 0.2) + (市场预期偏离度 × 0.15) + (评估者经验权重 × 0.15)
```

#### 权重说明
- **信息完整性 (25%)**：基础维度，信息是评估的前提
- **历史事件匹配度 (25%)**：核心维度，历史参照是评估的重要依据
- **多维度一致性 (20%)**：校验维度，确保评估的合理性
- **市场预期偏离度 (15%)**：调整维度，反映不确定性
- **评估者经验权重 (15%)**：主观维度，评估者的专业性

#### 权重调整原则
- **信息严重缺失**（信息完整性<0.4）：降低历史匹配度和多维度一致性的权重，提升信息完整性权重至40%
- **历史事件完全无参照**（历史事件匹配度<0.4）：降低历史匹配度权重至10%，提升多维度一致性权重至30%
- **事件完全超预期**（市场预期偏离度<0.4）：降低市场预期偏离度权重至5%，提升信息完整性权重至30%

### 置信度等级定义

#### 极高置信 (0.9~1.0)
**特征**：
- 信息极其完整
- 有高度相似的历史事件参照
- 多维度评估完全一致
- 符合市场预期

**示例**：OPEC常规产量调整（如50万桶/日增减产）

#### 高置信 (0.7~0.9)
**特征**：
- 信息较完整
- 有可参照的历史事件
- 多维度评估基本一致
- 与市场预期基本一致

**示例**：EIA库存数据超预期变化

#### 中等置信 (0.5~0.7)
**特征**：
- 信息一般
- 部分历史参照
- 多维度评估存在部分不一致
- 与市场预期有一定偏离

**示例**：地缘政治事件的潜在影响评估

#### 低置信 (0.3~0.5)
**特征**：
- 信息有限
- 缺乏历史参照
- 多维度评估存在矛盾
- 与市场预期偏离较大

**示例**：新兴技术对石油需求的长期影响

#### 极低置信 (0.0~0.3)
**特征**：
- 信息严重缺失
- 无历史参照
- 多维度评估严重不一致
- 完全超预期且缺乏依据

**示例**：突发未知事件的初步评估

### 置信度动态调整

#### 触发调整的情况
1. **多维度交叉验证失败**：置信度下调0.2-0.3
2. **发现历史事件评分异常**：置信度下调0.1-0.2
3. **市场反应与评分严重背离**：置信度下调0.1-0.15
4. **信息更新导致评估需修正**：置信度根据新信息调整

#### 调整后置信度下限
置信度下调后不得低于0.1，若计算值<0.1，则修正为0.1。

#### 调整记录要求
每次置信度调整必须在reasoning字段中记录：
- 调整原因
- 调整前置信度
- 调整幅度
- 调整后置信度

### 置信度评分自查清单

评估完置信度后，通过以下问题自查：

- [ ] 信息完整性评分是否真实反映了事件描述的详细程度？
- [ ] 历史事件匹配度评分是否有明确的历史事件参照？
- [ ] 多维度一致性评分是否基于实际的维度评估结果？
- [ ] 市场预期偏离度评分是否基于实际的市场预期和事件偏离？
- [ ] 评估者经验权重评分是否真实反映了对该类型事件的熟悉程度？
- [ ] 权重配置是否标准，如有调整是否有合理依据？
- [ ] 最终置信度是否与各维度评分和权重计算一致？
- [ ] 是否存在触发置信度动态调整的情况？

## 边界情况处理

### 情况1：事件已完全被市场预期
**判断标准**：
- 事件已在公开信息中讨论超过1周
- 主流分析师已有充分预期
- 市场价格已反映该预期

**处理方式**：
- 情绪极性：保持不变
- 冲击强度：下调0.2-0.4（取决于预期程度）

示例："市场已充分预期OPEC减产100万桶/日，会议正式公布"
- 基准：sentiment = 0.7, intensity = 0.7
- 调整：sentiment = 0.7, intensity = 0.4（下调0.3）

### 情况2：事件影响不确定或概率性
**判断标准**：
- 事件可能发生也可能不发生
- 影响规模和时间难以确定
- 存在较大的反预期可能

**处理方式**：
- 情绪极性：取最可能方向的极性，但幅度下调
- 冲击强度：基于预期影响的50-70%评分

示例："美国可能对伊朗实施新制裁，但还在讨论阶段"
- 基准：sentiment = 0.6, intensity = 0.6（假设制裁发生）
- 调整：sentiment = 0.4, intensity = 0.35（不确定性折扣）

### 情况3：多重事件同时发生
**判断标准**：
- 同一时间窗口内多个相关事件
- 事件方向可能相同或相反

**处理方式**：
- 情绪极性：加权平均（见Step 3）
- 冲击强度：取最大冲击事件强度，或略微上调（0.05-0.1）

### 情况4：事件影响逐渐消散
**判断标准**：
- 事件已经发生一段时间
- 市场关注度下降
- 影响已被消化

**处理方式**：
- 情绪极性：保持方向，但幅度递减
- 冲击强度：每递减一个评估周期，下调0.1-0.2
- 置信度：根据信息更新动态调整

### 情况5：多维度评估不一致
**判断标准**：
- 供给/需求/地缘/宏观多维度评估存在矛盾
- 不同维度指向不同的极性或强度

**处理方式**：
- 识别主导维度（影响规模最大、信息最明确的维度）
- 其他维度作为校验和参考
- 置信度下调0.1-0.2
- 在reasoning中详细记录不一致情况

示例："地缘冲突导致供给担忧，但经济衰退导致需求预期下降"
- 供给维度：sentiment = 0.6, intensity = 0.6
- 需求维度：sentiment = -0.5, intensity = 0.5
- 综合评估：sentiment = 0.1（小幅利多，供给主导）, intensity = 0.55
- 置信度：0.6（基准）- 0.1（不一致）= 0.5

### 情况6：历史事件评分异常
**判断标准**：
- 当前事件与历史事件高度相似，但市场反应差异显著
- 历史事件评分可能存在偏差或市场环境变化

**处理方式**：
- 分析市场环境差异（库存水平、需求增速、地缘格局等）
- 根据当前市场环境调整评分
- 置信度下调0.1-0.15
- 在reasoning中记录与历史事件的差异分析

## 评分自查清单

评估完事件后，通过以下问题自查：

- [ ] 是否明确识别了事件类型（供给/需求/地缘/宏观/其他）？
- [ ] 情绪极性的方向是否正确（供给减少→正，需求下降→负）？
- [ ] 情绪极性的幅度是否在合理范围内（-1.0~+1.0）？
- [ ] 冲击强度的评估是否考虑了规模、时间、关注度三个维度？
- [ ] 是否参考了历史相似事件的评分？
- [ ] 置信度各维度评分是否合理？
- [ ] 是否考虑了市场预期和不确定性因素？
- [ ] 最终评分是否与市场常识一致？
- [ ] 复合事件是否正确处理了多因素影响？
- [ ] 多维度评估不一致时是否正确处理？
- [ ] 是否执行了多维度交叉验证？
- [ ] 是否在reasoning中详细记录了评估依据？
